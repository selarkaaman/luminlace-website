# Lumin Lace — Women's Hair Product

Official website for Lumin Lace, a premium women's hair tools brand.

## Structure

```
├── index.html       # Main website
├── styles.css       # All styles
└── assets/
    └── logo.png     # Brand logo (place here)
```

## Setup

Open `index.html` in a browser. No build step required.

## Stack

- Vanilla HTML/CSS
- Google Fonts: Barlow Condensed + Barlow
- No dependencies
